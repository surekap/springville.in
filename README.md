# springville.in

Marketing website for the Springville real estate development.

## Overview

A public-facing marketing website for the Springville construction/real estate project. Contains property details, floor plans, image galleries, and contact forms for prospective buyers.

## Tech Stack

- HTML5 / CSS / JavaScript
- PHP (form/email handling)
- Static site with rich image assets

## Type

Internal public-facing website — Sureka/Hartex Group
